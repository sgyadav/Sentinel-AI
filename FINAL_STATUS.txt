📧 GMAIL SETUP (Optional)
═══════════════════════════════════════════════════════════════════════════════════

1. Go to Settings tab in SENTINEL AI
2. Fill in Gmail SMTP details:
   • SMTP Server: smtp.gmail.com
   • SMTP Port: 587
   • Email: your-email@gmail.com
   • App Password: 16-character Google App Password
   • Admin Email: recipient@example.com

3. Get App Password:
   → Go to Google Account Security
   → Enable 2-Factor Authentication
   → Generate App Password (select "Mail" + "Windows Computer")
   → Use the 16-character password

4. Click "Test Email" to verify configuration
